Usage

Simply integrate data from Tendril's API in to your Android Applications using the Cirrus Library. Just add a maven dependency, or reference our project from eclipse.

Usage »

Usage

First, you will want to go grab the library from github

git clone git@github.com:tendril-cirrus/4308Cirrus.git

The library contains four Primary Directories:

  1. tendril-domain
  2. tendril-android-lib
  3. tendril-example
  4. tendril-cirrus
Tendril Domain contains all of the domain models for objects returned from the API.
Tendril Android Lib is the core interface for making calls to the Tendril Connect API inside an android application.
Tendril Example is a bare bones Android Application that demonstrates how to make each and every call to the Tendril Connect API.
Tendril Cirrus is a more advanced example Android App that uses some geolocation features alongside the Tendril API.

Maven Integration

To use our library with your Maven Project, simply add the 4308-Cirrus project to your Maven Dependencies. (Don't forget to build the tendril-cirrus project after downloading it)

<dependency>
	<groupId>edu.colorado.cs.cirrus</groupId>
	<artifactId>tendril-android-lib</artifactId>
	<version>0.0.1-SNAPSHOT</version>
	<exclusions>
		<exclusion>
			<groupId>org.hamcrest</groupId>
			<artifactId>hamcrest-all</artifactId>
		</exclusion>
		<exclusion>
			<groupId>com.google.android</groupId>
			<artifactId>android</artifactId>
		</exclusion>
	</exclusions>
</dependency>

Eclipse Integration

We have included the necessary jars in the project directory. Simply add these to your project build path in Eclipse.

Tutorial

After setting up your project with Maven or Eclipse (see above), try creating a simple activity that extends AbstractAsyncActivity:

public class CirrusActivity extends AbstractAsyncTendrilActivity

Retrieve the tendril object:

tendril = TendrilTemplate.get();

Get some data:

user = tendril.fetchUser();
devices = tendril.getDevices():

More Information

You may have noticed that we are using strange SherlockActivities and things that reference actionbarsherlock. This is an open source library for adding Native Action Bar support to android versions below 4.0. Get more information about the library here: ActionBarSherlock
